# Nim Clock

A Cli Clock Written in Nim Language.

```txt
 ┏━┓   ╻        ┏━┓   ╻        ┏━┓  ┏━┓
   ┃   ┃    ╻     ┃   ┃    ╻     ┃  ┃
   ┃   ┃          ┃   ┃          ┃  ┃
 ┏━┛   ┃        ┣━┫   ┃        ┣━┫  ┗━┓
 ┃     ┃          ┃   ┃          ┃    ┃
 ┃     ┃    ╹     ┃   ┃    ╹     ┃    ┃
 ┗━━   ╹        ┗━┛   ╹        ┗━┛  ┗━┛
```

## Installation

```sh
git clone https://github.com/pysan3/nim_clock.git
cd nim_clock
nimble -d:release build
./nim_clock
```
